# `This code is beta and has no functions only the basics, if you want something that works you can use the hedystia.js npm.`

## Installation

```
npm i hedystia.js_lite

yarn add hedystia.js_lite

pnpm install hedystia.js_lite
```

## Nodejs Version

- `v18.6.0` or higher
